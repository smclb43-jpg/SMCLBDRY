AFTEM Mobile Station - CLEAN PACKAGE

This package is prepared with ASCII-only file and folder names.

How to use:
1. Upload ALL files and folders in this package to a NEW empty GitHub repository.
2. Do not rename anything.
3. In GitHub go to Actions > Build Android APK > Run workflow.
4. Download the artifact and install app-debug.apk on your phone.

Live usage:
1. Run AFTEM_MOBIL_SERVER_BASLAT.bat on the PC.
2. Phone and PC must be on the same Wi-Fi.
3. In the app enter API URL: http://PC_IP:8765
